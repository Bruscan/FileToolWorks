# FileToolWorks

Free online tools for file conversion and editing. Fast, simple, no signup required.

## About

FileToolWorks.com is a modern utility website offering free online tools for:
- Image conversion and editing
- PDF manipulation
- Video conversion
- And more

## Features

- Fast, client-side processing when possible
- No signup or account required
- Mobile-optimized
- Privacy-first (files deleted immediately)
- SEO-optimized for 2025

## Tech Stack

- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Turbopack

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
